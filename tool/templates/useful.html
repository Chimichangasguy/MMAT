Miscellaneous tools
VecScreen:

It screens your DNA sequence for potential vector sequence.
WebCutter:

It is an online tool for analyzing DNA sequences with restriction enzymes.
Tfsitescan:

This tool is intended for promoter sequence analysis and works best with sequences of ~500 nt.
ConSurf:

Server for the Identification of Functional Regions in Proteins.
RNAmmer

The server predicts 5s/8s, 16s/18s, and 23s/28s ribosomal RNA in full genome sequences.
InterProScan:

It is a resource that provides functional analysis of protein sequences by classifying them into families and predicting the presence of domains and important sites.
2ZIP:

Predicts Leucine Zippers in your amino acid sequence.
Jpred:

It is a Protein Secondary Structure Prediction server and has been in operation since approximately 1998.
PeptideMass:

It cleaves a protein sequence from the UniProt Knowledgebase (Swiss-Prot and TrEMBL) or a user-entered protein sequence with a chosen enzyme, and computes the masses of the generated peptides.
DisEMBL:

DisEMBL is a computational tool for prediction of disordered/unstructured regions within a protein sequence.
SignalP:

The SignalP 5.0 server predicts the presence of signal peptides and the location of their cleavage sites in proteins from Archaea, Gram-positive Bacteria, Gram-negative Bacteria and Eukarya. Protein sequences should be not less than 10 amino acids. The maximum number of proteins is 5000.
PolyPhen-2:

PolyPhen-2 (Polymorphism Phenotyping v2) is a tool which predicts possible impact of an amino acid substitution on the structure and function of a human protein using straightforward physical and comparative considerations.
PSAP:

A web based suite for protein structure analysis.
SAVES:

This meta server is designed to be a collective of results for various programs (ex. WHATCHECK, PROCHECK, ERRAT, Verify3D, PROVE).
SwissParam:

This service provides topology and parameters for small organic molecules compatible with the CHARMM all atoms force field, for use with CHARMM and GROMACS.
Agardir:

Agadir predicts the helical behaviour of monomeric peptides. It only considers short range interactions. Conditions such as pH, temperature and ionic strength are used in the calculation. Modifications of the termini are also allowed. Keep in mind that Agadir is not a program to predict secondary structure of proteins.
PDBePISA:

It is an interactive tool for the exploration of macromolecular interfaces.
CASTp:

CASTp identifies all pockets and voids on a protein structure and provides detailed delineation of all atoms participating in their formation.
DALI:

The Dali server is a network service for comparing protein structures in 3D. You submit the coordinates of a query protein structure and Dali compares them against those in the Protein Data Bank (PDB).
ProFunc:

The aim of the ProFunc server is to help identify the likely biochemical function of a protein from its three-dimensional structure.
ConSurf:

Server for the Identification of Functional Regions in Proteins.
Gene2Drug:

It ranks small molecules according to their ability to dysregulate an input set of pathways. Sets of pathways can be defined starting from a gene and exploiting its pathway annotations from a number of publicly available databases.
BLAST:

It finds regions of similarity between biological sequences. The program compares nucleotide or protein sequences to sequence databases and calculates the statistical significance.
ClustalO:

It is a new multiple sequence alignment program that uses seeded guide trees and HMM profile-profile techniques to generate alignments between three or more sequences.
T-Coffee:

It is a multiple sequence alignment program. Its main characteristic is that it will allow you to combine results obtained with several alignment methods.
